## What is Bool CLI?

Bool CLI is a powerful command-line tool for managing projects on [bool.com](https://bool.com). Deploy your code, manage versions, and collaborate with your team—all from your terminal.

Whether you need to ship code quickly with a single command or manage production deployments with full version control, Bool CLI provides a streamlined workflow for modern web development.

## Common workflows

### Quick deployment

Ship your code in seconds without any setup:

```bash
bool shipit
```

This command creates an anonymous Bool and deploys your files instantly. No API key needed.

### Production deployment

For authenticated deployments with full version control:

```bash
bool auth login
bool deploy --message "Add dark mode support"
```

### Managing projects

List, create, and manage your Bools:

```bash
bool list
bool create my-project
bool show my-project
```
